# 同类产品案例分析

每个文件分析一款游戏。重点不是介绍游戏，而是拆解它如何组织玩法系统。

## 两种写法

- 快速案例：使用 `research/templates/product-case-template.md`，适合记录一款游戏的核心机制和对本项目的启发。
- 深度拆解：使用 `research/templates/deep-product-analysis-template.md`，适合完整分析产品定位、体验、循环、架构、内容、经济、叙事和优化方案。

深度拆解的具体写作流程见 `deep-analysis-writing-system.md`。该流程以 `assets/标准化游戏拆解案框架.md` 的八大模块为主结构，要求最终转化为本项目的可验证设计假设或明确说明暂不转化。

当前已有：

- `yu-gi-oh.md`
- `magic-the-gathering.md`
- `pokemon-tcg.md`
- `balatro.md`
