# 拆解写作系统素材

本目录存放深度游戏拆解写作系统的本地参考素材。

## 标准结构

- `标准化游戏拆解案框架.md`：深度拆解案的八大模块结构标准。

## 示例素材

- `《明日方舟：终末地》基建系统拆解案.docx`：专项系统拆解示例，重点参考其围绕单一系统展开、再连接外部系统的写法。
- `游戏拆解--觅长生.docx`：完整产品拆解示例，重点参考其从核心定位、用户画像、核心循环到系统架构的展开方式。

## 使用规则

- 框架文件作为结构标准。
- 示例文件只作为写法参考，不直接替代本项目模板。
- 写入 `research/03-product-case-studies/` 的正式案例必须转化为本项目的设计启示、假设或暂缓问题。
